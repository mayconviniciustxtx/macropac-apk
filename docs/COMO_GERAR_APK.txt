MACROPAC RASTREAMENTO - COMO GERAR APK

1. Instalar Flutter no computador.
2. Abrir terminal dentro da pasta flutter_app.
3. Rodar:

flutter pub get
flutter build apk --release

4. APK gerado em:

build/app/outputs/flutter-apk/app-release.apk

URL API configurada:
https://mega4tech.com.br/macropac_rastreamento/api/
